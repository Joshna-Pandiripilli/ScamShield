# 🛡️ ScamShield

ScamShield is a Python-based web application that detects potentially fraudulent or phishing messages.

## 🚀 Features

* Analyze text messages for scam patterns
* Displays risk level and warnings
* Simple and user-friendly interface

## 🛠️ Tech Stack

* Python
* Streamlit

## 📌 Status

Currently a UI prototype. Detection logic will be added next.
